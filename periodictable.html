Highlightable Periodic Table

Here's a checklist of all the elements I've ever used in the course of my work as a chemist. After having a paper chart on my wall for a couple of years, I decided to make an offline script so that you can create your own highlighted periodic table easily in Microsoft Excel.

Download the file directly here. You can change the colours, but please follow the Instructions worksheet when you do, otherwise it gets confused! Contact me with any changes you'd like to make.
